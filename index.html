About Me

Hello everyone!

My full name is William Gilbert Go. Web design and development have always been my passion. My web development goal is to make applications easy to use and run smoothly across all types of devices and operating systems. I have high regards for the User Experience and User Interface principles, knowing that user satisfaction mostly relies on these principles. Right now I am completing my associate certificate in Applied Web Development at British Columbia Institute of Technology to be a full-stack developer of both PHP and Node.

Web Development

I truly enjoy building mobile-first single page applications using popular Javascript libraries. React is my most favorite Javascript library because it is object-oriented, which means that React allows us to build re-usable UI components. On the back end, Node JS is my expertise. This runtime environment technology allows developers to create highly-scalable web applications. For me, Node JS is an excellent technology to construct RESTful APIs.
